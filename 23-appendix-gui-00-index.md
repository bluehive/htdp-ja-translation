# 付録 D: Racket グラフィカルインタフェースツールキット

**原題:** The Racket Graphical Interface Toolkit  
**著者:** Matthew Flatt, Robert Bruce Findler, and John Clements  
**原本:** `extracted/appendix/gui/original_markdown_00_index.md`

```
+----------------------------+--------------------+
|  (require racket/gui/base) | package: `gui-lib` |
+----------------------------+--------------------+
+----------------------------+--------------------+
```

`racket/gui/base` ライブラリは、本マニュアルで定義されるクラス、インタフェース、手続きの束縛をすべて提供します。あわせて `racket/draw` および `file/resource` の束縛も含まれます。

```
+------------------+--------------------+
| #lang racket/gui | package: `gui-lib` |
+------------------+--------------------+
+------------------+--------------------+
```

`racket/gui` 言語は、`racket` 言語の束縛と、`racket/gui/base` および `racket/draw` モジュールの束縛をすべて合わせたものです。

`racket/gui` ツールボックスは、おおむね次の二つの部分に整理されています。

- **ウィンドウ機構ツールボックス（windowing toolbox）** — ウィンドウ、ボタン、メニュー、テキストフィールド、その他のコントロールを実装するため。
- **エディタ機構ツールボックス（editor toolbox）** — 従来型のテキストエディタ、テキストと図形を混ぜるエディタ、あるいは自由配置のレイアウトエディタ（ワープロ、HTML エディタ、アイコン型ファイルブラウザなど）を開発するため。

ツールボックスのどちらの部分も、`racket/draw` 描画ライブラリに大きく依存します。
